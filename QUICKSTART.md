## Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Run the Development Server
```bash
npm run dev
```

### Step 3: Open Your Browser
Navigate to `http://localhost:3000` (or the URL shown in your terminal)

## What You'll See

- A stunning 3D globe with glowing markers at conference locations
- Interactive particle background that follows your mouse
- Click any marker to view research papers from that location
- Smooth animations and futuristic design

- **Zoom**: Use mouse wheel
